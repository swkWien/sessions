The Background

  •  
  • This is part of a larger fantasy battle game
  • Players fight monsters using the equipment they are wearing.
  • This equipment will affect how much damage they do to the monster.
  • Monsters vary according to what armour and equipment they are wearing.
  • This part of the code is concerned with calculating how much damage a player is doing a monster when they fight.


The README contains example data to use for the test.
