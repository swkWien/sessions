Conflicting Refactorings

  •  
  • Programming in small groups
  • Free Language

The Definition:

according to Wikipedia, a Code Smell is any characteristic in the source code of a program that possibly indicates a deeper problem.

Two Code Smells:

  •  
  • Message Chain
  • Middle Man

Message Chain:

  •  
  • Some method in object A calls a method defined on an object C in a field of an object B.
  • Example: 
    	manager = person.getDepartment().getManager();
  • This increases coupling & violates the Law of Demeter.
  • To get rid of this smell, hide the delegate object.

Middle Man (Person?)

  •  
  • The only thing an object does is forward calls from one object to another one.
  • Example: 
    manager = person.getManager();
							
class Person {
	getManager(){
		return this.getDepartment().getManager();
	}
}
  • To get rid of this smell, call the target object directly, and remove the then useless middle man.

The dilemma:

To get rid of a message chain, you introduce a middle man.

To get rid of a middle man, you introduce a message chain.

The Exercise

  •  
  • Fantasy Battle

https://github.com/Neppord/FantasyBattle-Refactoring-Kata

The Background

  •  
  • This is part of a larger fantasy battle game
  • Players fight monsters using the equipment they are wearing.
  • This equipment will affect how much damage they do to the monster.
  • Monsters vary according to what armour and equipment they are wearing.
  • This part of the code is concerned with calculating how much damage a player is doing a monster when they fight.


The ReadMe contains example data to use for the test.

Round 1

  •  
  • Find a partner and agree on a programming language.
  • Write a test for the method calculateDamage()
  • Work deliberately, together, and test-driven!

Retrospective

  •  
  • What was your first test case?
  • What made it difficult to write tests?
  • Did you use mocks?

Round 2

  •  
  • Fix the message chains

Retrospective

  • Were you successful?
  • How did you solve the dilemma?
  • What was the most difficult part?
  • What did you learn?
