What even is ArchUnit?

https://www.archunit.org/

like unit tests, but for architecture
ArchUnit is a free, simple and extensible library for checking the architecture of your Java code using any plain Java unit test framework.
check dependencies between packages and classes, layers and slices, check for cyclic dependencies and more
